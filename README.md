# Number-Guessing-Game
It is a simple WPF project where the player has to guess a randomly generated number within a certain range. The game typically involves setting a range for the numbers and a certain number of attempts to guess the number. The game starts by generating a random number within the range specified, and then prompting the player to enter their guess. If the player's guess is too high or too low, the game will give a hint to help the player narrow down their next guess. The game continues until the player either guesses the number correctly or runs out of attempts.
